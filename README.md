Simple To-Do App

A simple and beginner-friendly To-Do List application built using HTML, CSS, and JavaScript.
This project helps users manage their daily tasks by adding and deleting items from a task list.

📌 Project Overview

The goal of this project is to practice basic web development concepts and understand how JavaScript interacts with the DOM (Document Object Model).
It allows users to add tasks, view them in a list, and remove tasks when completed.

✨ Features

Add new tasks

Delete existing tasks

Simple and clean user interface

Beginner-friendly project structure

🛠️ Technologies Used

HTML – for creating the structure of the application

CSS – for styling and layout

JavaScript – for implementing the application logic

📂 Project Structure
simple-todo-app
│
├── index.html # Main HTML file
├── style.css # Styling for the app
├── script.js # JavaScript logic
└── README.md # Project documentation
🚀 How to Run the Project

Download or clone the repository.

Open the project folder.

Double-click the index.html file.

The To-Do application will open in your web browser.

📸 Application Workflow

Enter a task in the input field.

Click the Add Task button.

The task will appear in the task list.

Click Delete to remove the task.
